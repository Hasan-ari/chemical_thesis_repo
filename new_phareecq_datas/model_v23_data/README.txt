-All the files inside the "output" folder contains PHREEQC output generated by a random input.

-The inputs can be found inside the "input" folder. Those inputs modify the PHREEQC model accordingly.

-Each output file contains the headers in the leading row.

-The output data is homogenous. It can be imported into a NumPy array.

-"main.py", "model_v23.phr", "phreeqc.dat" should reside in the same folder for program to work.

-"input", "output" and "plot" folders are generated at runtime.

-"default_output.txt" is necessary for "DEBUGGING" mode.

-Required libraries can be found inside the "requirements.txt"

-IPhreeqc should be downloaded from USGS website. The installation instructions can be found there as well.

-COM server of IPhreeqc should be faster than the DLL server.

-Randomizing any rock mineral randomizes the entire rock sample while maintaining the limit of 100 grams.

-The included output files tests "Alkalinity", "CO2", "H2S" and the rock sample.

-Default inputs and their randomization algorithms can be found in "generate_inputs()" in "main.py"

-The program uses every available thread except two unless "DEBUGGING". Be careful!
